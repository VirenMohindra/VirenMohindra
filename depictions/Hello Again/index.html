Auctor

Description

  • Auctor allows you to close the current app on lock, returning you to the homescreen. It also allows you to return to the first page if desired.

    This package functions as an iOS 9-compatible replacement for Eclair, although it was written independently.

Features

  • • Close current app on lock
    • Return to first page on lock
    • Prevent whitelisted apps from being closed
